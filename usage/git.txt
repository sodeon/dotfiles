Force empty folder into version control:
    In .gitignore:
        *
        !.gitignore

To merge disjointed branches (e.g. pull from new remote location)
    $git pull origin master --allow-unrelated-histories

Generate patch (for suckless):
    $git add files-to-be-patched
    $git commit
    $git format-patch --stdout HEAD^ > sxiv-andy-20190502.diff

Cherry pick:
    git cherry-pick -n # no commit. Always use this unless you want entire commit and no review

Discard unpushed local changes:
	git reset --hard <commit> # restore commit content to local files set HEAD memory to commit

Submodule:
    You will need this when pushing submodule for others: https://www.vogella.com/tutorials/GitSubmodules/article.html

Windows only:
    Add executable bit: git add --chmod=+x file_name


If .gitignore doesn't seem to work after modifying, run this scriot. (https://stackoverflow.com/questions/1139762/ignore-files-that-have-already-been-committed-to-a-git-repository)
    $git rm -r --cached .
