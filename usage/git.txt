Generate patch (for suckless):
    $git add files-to-be-patched
    $git commit
    $git format-patch --stdout HEAD^ > sxiv-andy-20190502.diff

Cherry pick:
    git cherry-pick -n # no commit. Always use this unless you want entire commit and no review

Discard unpushed local changes:
	git reset --hard <commit> # restore commit content to local files set HEAD memory to commit

Windows only:
    Add executable bit: git add --chmod=+x file_name
